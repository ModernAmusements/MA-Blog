# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-17

### Added
- Middleware to proxy migration (Next.js 16)
- DotMatrix animation speed control
- DotMatrix grid size presets
- DotMatrix color picker with preview
- New DotMatrix animations (spiral, sparkle, snake, bounce)
- SVG and sprite sheet export
- Image filters (edge detection, dithering, contrast)
- Contact API endpoint
- Newsletter API endpoint
- Focus-visible accessibility styles

### Fixed
- GIF export using gifenc library

### Changed
- CSS refactoring in DotMatrix module

## [0.9.0] - 2026-04-15

### Added
- Initial DotMatrix feature
- GIF export
- Multiple dot colors
