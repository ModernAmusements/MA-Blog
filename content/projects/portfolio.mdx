This is my personal portfolio website built from scratch.

## Tech Stack

- **Next.js 14** - App Router for modern React development
- **SCSS Modules** - Clean, scoped styling without Tailwind
- **MDX** - Markdown-based content for blog and projects
- **next-themes** - Dark/light mode support

## Features

- Fully responsive design
- Dark/light theme toggle
- Blog with tag filtering
- Project showcase
- SEO optimized
- RSS feed

## Challenges

The biggest challenge was keeping the design minimal while ensuring all features work seamlessly. I focused on:

1. Clean typography over complex layouts
2. CSS variables for easy theming
3. Static generation for optimal performance
